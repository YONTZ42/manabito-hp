# 合同会社マナビト WEBサイト 要件定義書・詳細設計書

---

## 0. ドキュメント概要

本ドキュメントは、合同会社マナビトのコーポレートサイト兼サービス紹介サイトを、  
**Next.js + React + Tailwind CSS** で構築するための **要件定義** と **詳細設計** を統合したものである。

本サイトは、地域密着型の信頼感と、先端行政事例・学び・人材育成を扱う専門性を両立し、  
行政・教育・企業・地域団体などの見込み顧客に対して、マナビトの理念、サービス、代表実績、活動情報、お問い合わせ導線を分かりやすく伝えることを目的とする。

---

# 1. 要件定義

## 1-1. プロジェクト概要

合同会社マナビトのWEBページを制作する。  
サイト全体は、**モバイル対応**、**SSRによるSEO配慮**、**地域密着型でやわらかな雰囲気** を前提とする。

参考デザインは主に以下とする。

- https://codateru.com/
- https://www.sarusho.com/
- https://civicpower.jp/#

特に `codateru.com` の以下の特徴を参考にする。

- 大きな文字画像を活かしたヒーロー表現
- 「〜したい」という目的別導線
- 問い合わせ導線の分かりやすさ
- やわらかく親しみやすいが、整理された情報設計

---

## 1-2. サイトの目的

- 合同会社マナビトの理念と価値を明確に伝える
- 行政・教育・企業・地域団体に向けて、課題とサービスの対応関係を分かりやすく示す
- 社長個人の実績を通じて信頼感を高める
- Instagram や最新情報を通して活動の継続性を伝える
- お問い合わせへの導線を強化する

---

## 1-3. 想定ターゲット

- 行政関係者
- 地方創生関連の担当者
- 教育機関関係者
- 社員研修・人材育成を検討する企業担当者
- 地域密着型の学習・体験プログラムを探している団体

---

## 1-4. サイトコンセプト

### キャッチコピー
**社会を、大きな教室に。**

### ブランドメッセージ
マナビトは『学び』と『人』を未来へつなぎ、行政支援や視察研修、体験学習、各種セミナーを通じて、子どもから大人まで、すべての『ヒト』に『マナビ』を届け、地域と人々の豊かな成長に貢献します。

### 印象キーワード
- 地域密着
- やさしい
- 誠実
- 見やすい
- 信頼感
- 行政・教育にも合う品の良さ
- 親しみやすいが幼すぎない

---

## 1-5. 技術要件

- フレームワーク: Next.js
- UIライブラリ: React
- スタイリング: Tailwind CSS
- レンダリング: SSR前提
- 対応端末: スマートフォン / タブレット / PC
- SEO配慮: metadata, OGP, semantic HTML, 見出し階層最適化

---

## 1-6. 初期実装範囲

初期フェーズでは、機能連携よりも **見た目・レイアウト・アニメーションの完成度** を優先する。

### 実装対象
- HTML / CSS / JavaScript による見た目の構築
- 各セクションのレイアウト実装
- 背景装飾やスライドなどのアニメーション実装
- ダミーデータによる表示
- ボタン、カード、見出しなど共通UIコンポーネント整備

### 初期実装対象外
- お問い合わせフォーム送信機能
- Instagram API 実連携
- CMS連携
- バックエンド処理
- 管理画面

---

## 1-7. 必要セクションと表示順

ページは上から以下の順で構成する。

1. ヘッダー
2. 理念＆キャッチコピー
3. みんなの悩み
4. ターゲットに合わせたサービス内容
5. 社長個人の実績
6. インスタ連携・最新情報
7. お問い合わせ
8. フッター

### この順番にする理由
最初に理念で世界観を伝え、その後に「悩み」→「解決策」→「信頼材料」→「活動実績」→「問い合わせ」と流すことで、訪問者が自然に理解しやすい構造になるため。

---

## 1-8. 機能要件

### ヘッダー
- ロゴを表示
- ボタン2つを表示
  - お問い合わせ
  - MENU
- 画面上部固定
- スクロール追従
- ボタンは共通 Button コンポーネントを利用する

### 理念＆キャッチコピー
- 1つのヒーローエリアとして配置
- デザイン文字画像 + テキスト
- 以下の文章を掲載する

> 社会を、大きな教室に。  
> マナビトは『学び』と『人』を未来へつなぎ、行政支援や視察研修、体験学習、各種セミナーを通じて、子どもから大人まで、すべての『ヒト』に『マナビ』を届け、地域と人々の豊かな成長に貢献します。

- 軽いアニメーションを許可する

### みんなの悩み
以下の内容を簡略化し、箇条書きまたはカード形式でわかりやすく提示する。

1. 子ども・若者の意見を社会に反映させたいが、どのように集約すればよいかわからない
2. 子ども・若者や地方創生に関する行政の先進事例を現地で知りたいが、行程作成や事業者との調整が煩雑
3. 地方での体験学習や社員研修・人材育成を考えているが、充実した計画を立てる時間がない / ノウハウがない / 連絡の取り方がわからない

### ターゲットに合わせたサービス内容
- 横スライドで表示
- 自動横スライド
- カードコンポーネントを共通化
- カードの構成は以下
  - 画像エリア: 70%
  - テキストエリア: 30%
- 可変項目
  - 画像
  - 色
  - タイトル
  - 説明

想定表示テーマ:
- 子ども・若者からの意見を聞きたい
- 行政の先進事例やサービスを知りたい
- 専門家から専門知識を学びたい
- 地域密着の体験学習・社員研修や人材育成をしたい

※ 列挙内容は4テーマだが、設計は将来拡張を見込み、5件以上に対応できる配列駆動とする。

### 社長個人の実績
- 明るく、地域密着型の印象で紹介する
- 初期はデモ用のプレースホルダー表示
- 後からプロフィールや実績カードに差し替え可能な構造にする

### インスタ連携・最新情報
- 初期はダミーカード表示
- 将来的に Instagram API で最新投稿一覧を表示
- 自社最新情報も同じセクション内で表示できるようにする

### お問い合わせ
- ボタン2つ
  - 電話番号
  - フォームお問い合わせ
- 両方とも共通 Button コンポーネントを使用
- 電話番号ボタンは `tel:` リンク
- フォームボタンはモーダル表示
- 初期はモーダルの見た目のみ実装し、送信機能は未実装とする

---

## 1-9. 非機能要件

### SEO
- SSRで初期レンダリング
- title / description / OGP / canonical 設定
- 見出し階層の最適化
- テキスト主体の構成
- 画像に alt を付与

### パフォーマンス
- 画像最適化
- 過剰なJavaScriptを避ける
- アニメーションは軽量に抑える
- LCP改善のためヒーロー領域を優先最適化

### アクセシビリティ
- 十分なコントラスト
- 押しやすいボタンサイズ
- キーボード操作考慮
- aria-label 適切設定

### レスポンシブ
- スマホ最優先
- タブレット対応
- PC対応
